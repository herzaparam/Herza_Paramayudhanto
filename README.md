# herza_paramayudhanto
## react.js app

an apps which can find github user and see a list of their repository.

## Getting Started
### installation
- clone the repo
```sh
git clone https://github.com/herzaparam/herza_paramayudhanto.git
```
- install NPM packages
```sh
npm install
```

## Run the apps
- start in local
```sh
npm start
```
- build production
```sh
npm run build-prod
```
